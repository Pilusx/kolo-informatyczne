4. Iterowanie po strukturze (* Wersja niedopracowana, jeszcze będzie ulegać zmianom…)
Możemy to robić dla vectora, listy, zbiorów oraz map. (Te kontenery które mają funkcje begin oraz end)
a) Wersja I
T t;
….
for(T::iterator it = t.begin(); it != t.end(); ++it) {
… zrob cos(*it);
}

b) Wersja II
Lub prościej, jeśli tylko odczytujemy dane ze struktury.
for(auto v : t) {
… zrob coś(v);
}

c) Wersja III
Jeśli chcemy ją zmodyfiować, musimy uzyskać referencję na obiekt.
for(auto &v : t) {
… zrób coś v
}
