Część 1) void potegi(unsigned x)

Mamy do dyspozycji globalną tablicę unsigned pot_n[N], której rozmiar ustalamy zmienną globalną const int N;
Funkcja ma ustawiać zmienną globalną n na x. (tj n = x), a następnie wypełnić tablicę pot_n kolejnymi potęgami liczby n. tj. {1, n, n2, ... , nN-1}
Oczekiwana złożoność O(N)

Przykład 1.

Tj. po wywołaniu funkcji potegi(2) tablica ma zawierać elementy {1, 2, 4, 8, 16 ... }

Część 2a) unsigned wyciagnij(unsigned x, unsigned k)

Mając daną liczbę n zapisaną globalnie.
Zakładamy, że została wcześniej wykonana funkcja potegi(n).
Napisać funkcję wyciagnij(x,k), która zwraca k-tą "cyfrę" liczby x w zapisie n-kowym.
Oczekiwana złożoność O(1)

Przykład 1.

Wywołujemy funkcję potęgi(2)
// 13 = 1101 (w systemie dwójkowym tj. 8 + 4 + 0*2 + 1)
wyciagnij(13, 0) zwraca 1
wyciangij(13, 1) zwraca 0
wyciagnij(13, 2) zwraca 1
wyciagnij(13, 3) zwraca 1
wyciagnij(13, 4) zwraca 0

Przykład 2.

Wywołujemy funkcję potęgi(3)
// 17 = 122 (w systemie trójkowym tj. 9 + 2*3 + 2*1)
wyciagnij(17, 1) zwraca 2

Część 2b) void wypisz(unsigned x)

Mamy daną liczbę n zapisaną globalnie.
Zakładamy, że została wcześniej wykonana funkcja potegi(n).
Napisać funkcję wypisz(x), która wypiszę liczbę w systemie n-kowym.
Kolejne "cyfry" proszę oddzielać spacją.
Oczekiwana złożoność O(N)

Przyklad 1.

Gdy N = 9
Po wywołaniu potęgi(3), wypisz(17) powinno wypisać:
0 0 0 0 0 0 1 2 2

Część 3a) void generuj(unsigned x)

N = 9
Mamy vector vectorów (vector<vector<unsigned> > g) trzymany globalnie. Przed wywołaniem funkcji generuj należy wywołać funkcję potegi(3) i rozszerzyć vector g do rozmiaru 39 (resize)
Napisać funkcję generuj(x), która dla danej liczby x zapisuje do vectora g[x] (w dowolnej kolejności) wszystkie liczby, które spełniają jednocześnie wszystkie poniższe warunki
  • długość jej zapisu w systemie trójkowym nie przekracza 9 (<= 9)
  • można ją utworzyć poprzez zmianę jednego zera (0) w zapisie 3-kowym na jedynkę (1) lub dwójkę (2)

Część 3b) void wypisz_sasiadow(unsigned x)

Należy napisać funkcję, która wypisuje zawartość wektora g[x] w formacie podanym w Zadaniu 1 na tej stronie.

Przykład 1.

generuj(0) powinna wypisać:
0 : 1 3 9 27 81 243 729 2187 6561 2 6 18 54 162 486 1458 4374 13122

Część 4) unsigned zmien_gracza(unsigned x)

Funkcja ma zwracać:
2, gdy x == 1
1, gdy x == 2
W pozostałych przypadkach może zwracać cokolwiek.

Funkcja main, przykładowe użycie powyższych funkcji

int main() {
	potegi(2);
	for(int i = 0; i <= 4; i++)
		cout << wyciagnij(13, i) << endl;
	cout << endl;
	
	potegi(3);
	wypisz(17);
	g.resize(3*pot_n[8]);
	generuj(0);
	wypisz_sasiadow(0);
	
	return 0;
}

Rozwiązanie części 1-4

Proszę nie czytać kodu jeśli nie jest to bezwzględnie konieczne, tzn. po wielu nieudanych próbach samodzielnego napisania

Wstęp do części dalszej

Gramy w kółko i krzyżyk.
Pierwszy stawiamy krzyżyk (1), a następnie kółko (2), kolejne ruchy wykonywane są na zmianę.
Wyobraźmy sobie, że mamy jakieś ustawienie kółek i krzyżyków np. takie:
X O
X
O X

Możemy je przekształcić w liczbę w następujący sposób.
Puste pole to 0, X to 1, O to 2
1*30 2*31 0*32
0*33 1*34 0*35
2*36 0*37 1*38
Otrzymujemy wówczas liczbę 102010021 w zapisie trójkowym (tj. 8107), która jednoznacznie określa nam stan gry.
Co więcej:
Postawienie krzyżyka na planszy jest równoznaczne z dodaniem czynnika 1*3i
Postawienie kółka na planszy jest równoznaczne z dodaniem czynnika 2*3i
